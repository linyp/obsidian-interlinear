Плагин для **параллельного перевода по абзацам** в режиме чтения
[Obsidian](https://obsidian.md). Откройте заметку на иностранном языке и нажмите
кнопку перевода — Interlinear покажет русский или другой выбранный перевод сразу
под каждым исходным абзацем. Можно просматривать исходный текст вместе с переводом
или только перевод.

> Интерфейс плагина пока доступен на английском языке. Названия настроек и команд
> в этом README совпадают с английским интерфейсом, чтобы их было проще найти.

## Безопасность по умолчанию

- **Никогда не изменяет заметки.** Перевод добавляется только в DOM на уровне
  отображения. После закрытия и повторного открытия заметки Markdown-файл на диске
  остается неизменным.
- **Никогда не переводит автоматически.** Открытие и переключение заметок, прокрутка,
  изменение макета или настроек не отправляют запросы на перевод. Перевод начинается
  только после явного действия через плавающую кнопку, строку состояния или палитру команд.
- **BYOK и отсутствие телеметрии.** API-ключи и учетные данные приложений хранятся
  только в настройках плагина внутри хранилища и отправляются лишь выбранному сервису перевода.
- **Только режим чтения.** Режим редактирования и Live Preview не поддерживаются.

## Основные возможности

- **Перевод всей заметки одним нажатием.** На компьютере используйте строку состояния,
  на мобильном устройстве — плавающую кнопку в правом нижнем углу. Во время перевода
  отображается прогресс по пакетам (`3/12` и т. п.).
- **Два режима отображения.** Мгновенно переключайтесь между двуязычным режимом
  (оригинал + перевод) и режимом только перевода без повторного запроса. В режиме
  только перевода наведите указатель или коснитесь перевода, чтобы увидеть оригинал.
- **Пять стилей оформления.** Рамка, цитата, приглушенный текст, пунктирное подчеркивание
  и учебная маска переключаются только средствами CSS.
- **Постоянный кэш переводов.** Результаты индексируются по хэшу содержимого и хранятся
  в `cache.json` в папке плагина. Повторный перевод выполняется быстрее и дешевле;
  исходный текст в кэш не записывается.
- **Поддержка виртуализированного отображения Obsidian.** Видимые абзацы переводятся
  сразу, остальные заранее сохраняются в кэш и вставляются при прокрутке.
- **Пропуск неподходящего содержимого.** Код, формулы, блоки только с изображениями,
  URL, символами или цифрами и текст, который можно надежно распознать как уже написанный
  на целевом языке, не переводятся.
- **Сменные серверные сервисы.** Поддерживаются DeepSeek, OpenAI, SiliconFlow, Ollama,
  пользовательские OpenAI-совместимые endpoints, Baidu Translate (百度翻译) и Youdao
  (有道智云). Все сетевые запросы выполняются через `requestUrl` Obsidian.
- **Предустановленные целевые языки.** Доступны русский (`ru`), японский (`ja`),
  корейский (`ko`), вьетнамский (`vi`), упрощенный и традиционный китайский, английский
  и другие языки. Можно также ввести собственный код языка.

## Сеть, учетные записи и конфиденциальность

- Плагин отправляет переводимые абзацы выбранному сервису только после явного запуска
  перевода или нажатия **Test connection**.
- Вы предоставляете собственный API-ключ или учетные данные приложения. Плату взимает
  выбранный провайдер, а не Interlinear.
- Телеметрия и аналитика отсутствуют.
- Настройки хранятся в `data.json`, резервная копия перед миграцией может находиться в
  `data.backup.json`, а кэш переводов — в `cache.json`. Все файлы расположены в папке плагина.
- При синхронизации хранилища учетные данные также могут попасть на другие устройства.
  Если хранилище управляется через Git, добавьте как минимум
  `.obsidian/plugins/interlinear/data.json` и
  `.obsidian/plugins/interlinear/data.backup.json` в `.gitignore`.

## Установка

### Из Obsidian (рекомендуется)

1. Откройте **Settings → Community plugins → Browse**.
2. Найдите **Interlinear**, затем выберите **Install** и **Enable**.

Можно также открыть [страницу плагина](https://obsidian.md/plugins?id=interlinear)
и нажать **Install**.

### Обновление с v0.2.5 до v0.3.0

В v0.3.0 используется settings schema v2. Плоские настройки v0.2.5 мигрируются
один раз; перед перезаписью `data.json` исходные данные сохраняются в
`data.backup.json`.

Если настройки плагина синхронизируются, обновите Interlinear на **всех синхронизируемых
устройствах до изменения любых настроек**. Одновременная работа разных версий и
откат после миграции не поддерживаются.

### BRAT / ручная установка

- Чтобы получать версии до их публикации в официальном каталоге, установите
  [BRAT](https://github.com/TfTHacker/obsidian42-brat), выполните команду
  **BRAT: Add a beta plugin for testing** и укажите `linyp/obsidian-interlinear`.
- Для ручной установки скачайте `main.js`, `manifest.json` и `styles.css` из
  [последнего релиза](https://github.com/linyp/obsidian-interlinear/releases/latest)
  и поместите их в `<your-vault>/.obsidian/plugins/interlinear/`.

## Настройка

Откройте **Settings → Interlinear**.

| Настройка | По умолчанию | Описание |
| --- | --- | --- |
| Service | DeepSeek | Выберите LLM или традиционный машинный перевод. У каждого пресета собственные учетные данные и расширенные настройки. |
| API key _(только LLM)_ | _пусто_ | API-ключ выбранного LLM-сервиса. |
| App ID + secret _(Baidu / Youdao)_ | _пусто_ | Учетные данные из консоли разработчика сервиса. |
| Base URL _(только LLM)_ | `https://api.deepseek.com` | OpenAI-совместимый endpoint. |
| Model _(только LLM)_ | `deepseek-v4-flash` | Используемая модель. |
| Test connection | — | Отправляет небольшой запрос для проверки учетных данных и соединения. |
| Target language | `zh-CN` | Выберите `ru`, `pt-BR`, `es`, `zh-TW` или укажите собственный код языка. |
| Default display mode | Bilingual | Режим отображения после первого перевода. |
| Translation style | Border | Визуальный стиль перевода. |
| Floating button | Mobile only | Always / mobile only / never. |
| Concurrency | 10 | Максимальное число одновременных запросов. |
| Min interval (ms) | 0 | Минимальный интервал между запусками запросов. |
| Max retries | 3 | Число повторов при ошибках 429 или временных сбоях. |
| Batch char budget | 4000 | Максимальное число символов в одном запросе. |
| Max segments per request | 12 | Максимальное число блоков в одном запросе. |
| Custom instructions _(только LLM)_ | _пусто_ | Терминология, стиль или предметные инструкции для prompt. Содержимое входит в идентификатор кэша. |
| Persistent cache | On | Сохраняет кэш переводов между перезапусками. |

## Использование

1. Откройте заметку в **reading view**.
2. На компьютере нажмите **Translate** в строке состояния; на мобильном устройстве
   нажмите плавающую кнопку в правом нижнем углу.
3. Нажмите еще раз, чтобы переключиться между переводом и оригиналом.
4. Используйте кнопку режима для переключения между **bilingual** и
   **translation-only**. Новый запрос на перевод не отправляется.

В палитре команд доступны следующие команды. Сочетания клавиш по умолчанию не назначены.

- **Interlinear: Translate / show original**
- **Interlinear: Toggle display mode (bilingual / translation-only)**
- **Interlinear: Clear translations**

## Ограничения

- Поддерживается только режим чтения; режим редактирования и Live Preview не поддерживаются.
- Списки и таблицы переводятся как единый блок. Плоские списки восстанавливаются,
  но вложенная структура сохраняется не полностью.
